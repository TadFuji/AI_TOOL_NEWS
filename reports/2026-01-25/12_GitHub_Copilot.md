- **Date**: 2026-01-22
- **URL**: https://x.com/github/status/2014382805690487290
- **Summary**: GitHub Copilot SDKがリリース。Copilot CLIのエージェントコアを任意のアプリケーションに数行のコードで埋め込み可能。多モデルセッション、カスタムツール、GitHub認証などをサポート。
