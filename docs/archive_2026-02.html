GitHub Copilot
2026年02月01日 00時50分

GitHub Copilot CLIにおいて、利用するLLM（大規模言語モデル）を自由に選択・切り替えられる新機能が追加されました。新しく導入された「/model」や「/plan」といったコマンドを使用することで、実行するタスクに最適なモデルを即座に使い分けることが可能です。

GitHub Copilot PICK UP
2026年02月01日 00時50分

GitHub Copilot CLIにて、利用するモデルを自由に切り替えられる新機能が追加されました。新しく導入された「/model」や「/plan」コマンドを使用することで、作業内容に応じて最適なLLMを選択して実行することが可能です。

用途に合わせて最適なモデルを選択できるため、コマンド生成の精度向上や、より複雑な開発タスクへの柔軟な対応が可能になります。
