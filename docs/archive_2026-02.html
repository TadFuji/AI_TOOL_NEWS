GitHub Copilot PICK UP
2026年02月01日 00時50分

GitHub Copilot CLIにおいて、使用するLLMを自由に選択・変更できる機能が導入されました。`/model`や`/plan`といったコマンドを使用することで、実行するタスクに合わせて最適なモデルを柔軟に切り替えられます。

タスクの特性に応じて最適なモデルを選択できるため、CLI環境における回答の精度向上や開発プロセスの最適化が期待できます。
