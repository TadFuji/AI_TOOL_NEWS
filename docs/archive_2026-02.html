GitHub Copilot PICK UP
2026年02月01日 00時50分

GitHub Copilot CLIにおいて、利用するLLM（大規模言語モデル）を自在に切り替えられる機能が追加されました。`/model`や`/plan`といったコマンドを活用することで、実行したいタスクの内容に合わせて最適なモデルをユーザーが選択し、柔軟に使い分けることが可能です。

単一のモデルに依存せず、タスクの複雑さや目的に応じてモデルを最適化できるため、開発効率と回答精度の向上が見込めます。
