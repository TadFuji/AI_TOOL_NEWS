GitHub Copilot PICK UP
2026年02月01日 00時51分

GitHub Copilot CLIにて、複数のLLMをタスクに応じて選択・切り替えが可能になりました。新コマンド「/model」や「/plan」を使用することで、特定の作業に最適なモデルを自在に選べるようになり、開発の柔軟性が大きく向上しています。

タスクの特性に合わせて最適な推論モデルを使い分けられるため、回答の精度向上や処理の効率化をユーザー自身で制御できるようになります。
