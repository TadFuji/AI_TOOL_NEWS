GitHub Copilot
2026年02月01日 00時50分

GitHub Copilot CLIにおいて、使用するLLM（大規模言語モデル）を柔軟に選択・切り替えられる機能が追加されました。`/model`コマンドでモデルを切り替えたり、`/plan`コマンドで実行計画を確認したりすることが可能です。

タスクの特性に合わせて最適なモデルを選択できるようになったことで、コマンド生成の精度向上や開発ワークフローの柔軟性が高まります。

GitHub Copilot PICK UP
2026年02月01日 00時50分

GitHub Copilot CLIにて、利用するモデルを自由に切り替えられる新機能が追加されました。新しく導入された「/model」や「/plan」コマンドを使用することで、作業内容に応じて最適なLLMを選択して実行することが可能です。
